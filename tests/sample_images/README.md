# sample_images/

Place your test invoice and medicine strip images here.

## Required for Test Phase 2 (Extractor Test):

- `test_invoice.jpg` — A clear, well-lit photo of a medical invoice + medicine strip.

## Tips for a Good Test Image:
- Use good lighting (no shadows, no glare)
- Both the invoice header (vendor name, invoice number) and medicine label (batch, expiry) should be in the same frame
- Minimum 800×600 px (higher is better for OCR accuracy)

## After Camera Test:
The camera test script auto-saves a snapshot here as:
- `camera_test_snapshot.jpg`
